# Agent Context Ops v0.2.1

v0.2.1 is a small patch release focused on release hygiene.

## Highlights

- Package version bumped to `0.2.1`.
- Changelog and roadmap updated for the patch release.
- The `v0.2.0` release commit was republished without an unintended co-author trailer.

## Upgrade

From a local checkout:

```bash
git pull
python -m pip install -e .
agent-context-ops --version
```

Expected output:

```text
agent-context-ops 0.2.1
```
