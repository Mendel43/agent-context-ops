# Publishing

## Local repository

```bash
git init
git add .
git commit -m "Initial Agent Context Ops toolkit"
```

## GitHub

Create a public GitHub repository named:

```text
agent-context-ops
```

Then:

```bash
git remote add origin https://github.com/YOUR_USER/agent-context-ops.git
git branch -M main
git push -u origin main
```

## Before applying to Codex for OSS

- Confirm the repository is public.
- Confirm CI is green.
- Add a screenshot or short GIF of the CLI output.
- Add a few real issues or roadmap tasks.
- Make at least a couple of focused commits, not one huge unreviewable dump.
